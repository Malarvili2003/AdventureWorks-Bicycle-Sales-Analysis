# AdeventureWorks-Bicycle-Sales Data Visualization Project using Power BI

## Project Overview

This project focuses on analyzing the sales performance of AdventureWorks, a bicycle manufacturing company that sells mountain, road, and touring bikes along with related components and accessories. The goal was to transform raw transactional records into meaningful business insights using Power BI and build an interactive dashboard that supports data-driven decision-making.

The dataset was cleaned and transformed using Power Query to ensure accuracy, consistency, and proper formatting across sales, product, customer, and regional data. A structured data model was created by establishing relationships between multiple tables, enabling accurate aggregation and cross-dimensional analysis. DAX measures were developed to calculate key business metrics such as total revenue, total profit, profit margin percentage, year-to-date sales, and growth trends. These measures enhanced analytical flexibility and ensured reusability across the report.

The final dashboard presents revenue trends over time, product category performance, regional sales distribution, and top-performing products through interactive visuals and slicers. The report was published to Power BI Service, allowing users to explore insights dynamically and gain a clear understanding of business performance.
