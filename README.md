# AdventureWorks Bicycle Sales Data Visualization Project (Power BI)

## Project Overview

This project analyzes the sales performance of AdventureWorks, a bicycle manufacturing company that sells mountain, road, and touring bikes along with related components and accessories.

The objective was to transform raw transactional data into meaningful business insights using Microsoft Power BI and build an interactive dashboard that supports data-driven decision-making.

The dataset was cleaned and transformed using Power Query to ensure accuracy, consistency, and proper formatting across sales, product, customer, and regional data.

A structured data model was created by establishing relationships between multiple tables, enabling accurate aggregation and cross-dimensional analysis.

DAX measures were developed to calculate key business metrics such as:

- Total Revenue  
- Total Profit  
- Profit Margin (%)  
- Year-to-Date (YTD) Sales  
- Sales Growth Trends  

These measures improved analytical flexibility and ensured reusability across the report.

---

## Dashboard Features

- Revenue trends over time  
- Product category performance analysis  
- Regional sales distribution  
- Top-performing products  
- Profit margin insights  
- Interactive slicers and filters  

---

## Tools & Technologies Used

- Microsoft Power BI  
- Power Query (ETL & Data Cleaning)  
- DAX (Data Analysis Expressions)  
- Data Modeling (Star Schema)  

---

## Project Outcome

The final dashboard presents revenue trends over time, product category performance, regional sales distribution, and top-performing products through interactive visuals and slicers. The report was published to Power BI Service, allowing users to explore insights dynamically and gain a clear understanding of business performance.
